# Trip Nexus Platform

A full stack travel and tour booking web application built using Node.js, Express.js, MongoDB, and JavaScript.

## Features

- User authentication and authorization
- Tour listing and detailed pages
- Interactive maps using Mapbox
- Responsive design
- REST API integration
- MongoDB database connection
- MVC architecture

## Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose
- JavaScript
- HTML
- CSS
- Mapbox
- Pug Template Engine

## Project Structure

```bash
controllers/
models/
routes/
public/
views/
dev-data/
server.js
app.js
```

## Installation

1. Clone the repository

```bash
git clone https://github.com/Govindaraj773/trip-nexus-platform.git
```

2. Navigate to project folder

```bash
cd trip-nexus-platform
```

3. Install dependencies

```bash
npm install
```

4. Create config.env file

```env
DATABASE=your_mongodb_connection
DATABASE_PASSWORD=your_password
JWT_SECRET=your_secret
```

5. Start the server

```bash
npm start
```
